- Variable declarations with `var`
- Arithmetic expressions: +, -, *, /, ()
- Printing with `print`


Statements:
    var <identifier> = <expression>
    print <expression>

Expressions:
    <number>
    <identifier>
    (<expression>)
    <expression> +|-|*|/ <expression


Program 1:
var a = 3
var b = a + 4 * 2
print b

Program 2:
var x = (1 + 2) * 5
print x


var, print


- if/else
    var x = 5
    if x > 3
        print "x is big"
    else
        print "x is small"
- while loops
    var i = 0
    while i < 5
        print i
        var i = i + 1

- for loops
    for var i = 0; i < 5; i = i + 1
        print i
- functions
- comparison operators: >, <, ==
